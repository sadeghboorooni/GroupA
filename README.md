تبلیغ نما یک پروژه ی درسی است که توسط دانشجویان دانشکده ی کامپیوتر دانشگاه علم و صنعت شروع شده است. این پروژه از یک تکلیف درسی برای درس مهندسی نرم افزار 2 شروع شده و بعد از آن نیز ادامه خواهد داشت!
